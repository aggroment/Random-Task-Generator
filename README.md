## Краткое описание

Консольное приложение для генерации случайных задач с возможностью фильтрации, сохранения истории и управления задачами. Приложение использует паттерн Factory для создания задач разных типов и сохраняет историю в JSON файл.

## Функциональность

- ✅ Генерация случайных задач (Работа, Спорт, Учеба, Другое)
- ✅ Добавление пользовательских задач с валидацией
- ✅ Хранение истории задач (очередь из 100 последних задач)
- ✅ Фильтрация по типу и сложности
- ✅ Сохранение/загрузка истории в JSON
- ✅ Очистка истории

## Требования

- Python 3.7+
- Стандартные библиотеки (random, json, collections, enum, dataclasses)
