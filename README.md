## Описание программы
Random Task Generator — это консольное приложение для генерации случайных задач разных типов. Программа помогает разнообразить повседневные дела, предлагая случайные задачи из различных категорий (работа, спорт, учеба, дом, хобби).

### Особенности:
- 🎯 Генерация случайных задач с учётом типа и сложности
- 📋 История всех сгенерированных задач (очередь из 100 записей)
- 🔍 Фильтрация задач по типу и сложности
- ➕ Добавление пользовательских задач
- 📊 Статистика по типам и сложности задач
- 💾 Сохранение и загрузка истории в JSON
- 🏗️ Паттерн Factory для создания задач разных типов
- 🧬 Полностью ООП-архитектура (инкапсуляция, наследование, полиморфизм)

## Как запустить приложение

### Требования
- Python 3.6 или выше
- Стандартные библиотеки (random, json, collections, os, abc)

# 2. Откройте терминал в папке с файлом
